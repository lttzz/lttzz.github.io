benchmark

In computing, a benchmark is the act of running a computer program, a set of programs, or other operations, in order to assess the relative performance of an object, normally by running a number of standard tests and trials against it.[1] The term benchmark is also commonly utilized for the purposes of elaborately designed benchmarking programs themselves.

看到csapp优化程序性能那一节，觉得有些结果反直观，我还是愿意动手跑一下例子，写for(int i = 0; i < MAX; ++i) 配上linux下的 time 次数多了也麻烦，而且不够逼格，搜到了google开源的C++ benchmark库，遂学习一下。

#include <benchmark/benchmark.h>

// 要进行benchmark的函数
void func(void)
{
    for (int i = 0; i < 1e5; ++i)
        ;
}

static void BM_Func(benchmark::State& state)
{
    for (auto _ : state)
        func();
}
// Register the function as a benchmark
BENCHMARK(BM_Func);

// Run the benchmark
BENCHMARK_MAIN();

#include <benchmark/benchmark.h>
#include <iostream>

using namespace std;

void func(int u, int v)
{
    cout << u << " " << v << endl;
}
static void BM_Func(benchmark::State& state)
{
    for (auto _ : state)
        // state.range(index)
        func(state.range(0), state.range(1));
}

// 传递多个参数使用Args({})，单个参数用Arg()
BENCHMARK(BM_Func) -> Args({1, 2});

BENCHMARK_MAIN();

#include <benchmark/benchmark.h>
#include <iostream>

using namespace std;

void func(int u, int v)
{
    cout << u << " " << v << endl;
}
static void BM_Func(benchmark::State& state)
{
    for (auto _ : state)
        func(state.range(0), state.range(1));
}

// 传递的参数是集合{0， 100， 1000}和集合{1， 2}的笛卡尔积
// 即{{0, 1}, {0, 2}, {100, 1}, {100, 2}, {1000, 1}, {1000, 2}}
BENCHMARK(BM_Func) -> RangeMultiplier(100) -> Ranges({{0, 1000}, {1, 2}});

BENCHMARK_MAIN();

烟酰胺

知乎：毁三观！烟酰胺美白，知道真相的我眼泪流下来…

好奇所谓冻疮膏“治疗”冻疮的原理，百度百科看了眼冻疮病因 冷暖急变，局部小动脉发生收缩，久之动脉血管麻痹而扩张，静脉淤血，局部血液循环不良而发病，又瞄了一眼下面的治疗方法，口服烟酰胺、硝苯地平等血管扩张剂。

烟酰胺？？？血脉扩张剂？？？

曾经看过混知健康推文“被晒黑了，咋样才能白回来？”一文 全是图，文中提到美白产品中主要是三类功效的成分：

  • 熊果苷和鞠酸（曲酸）：阻止络氨酸形成中间体
  • 维C和谷胱甘肽：阻止中间体变成黑色素
  • 烟酰胺：阻止黑色素的转移

搜索烟酰胺就搜到了开头知乎晓安的文章，时间早于陈磊。关于烟酰胺美白原理，晓安考证是宝洁公司在论文中首次提到，但实验没说服化工硕士，遂深扒了烟酰胺美白这一说法，查文献，做实验，一套流程下来：烟酰胺可以剥！脱！角！质！且效果媲美果酸！且刺激性比果酸/水杨酸小很多。

真假不知道，但是这严谨程度说服我了，而且用烟酰胺没觉得变白之类，控油祛痘效果明显。
